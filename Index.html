#if指令

  • 一个Vue的实例其实正是一个MVVM模式中的ViewModel-因此后可经常使用vm这个变量名
  • 在实例化Vue时,需要传入一个选项对象，它可以包含数据、模板、挂载元素、方法、生命周期钩子等
  • 可以扩展Vue构造器，从而用预定义的选项创建可复用的组件构造器
  • 组件其实都是被扩展的Vue实例

属性与方法

                
                    var data = { a: 1 }
                    var vm = new Vue({
                    data: data
                    })

                    vm.a === data.a // -> true

                    // 设置属性也会影响到原始数据
                    vm.a = 2
                    data.a // -> 2

                    // ... 反之亦然
                    data.a = 3
                    vm.a // -> 3
